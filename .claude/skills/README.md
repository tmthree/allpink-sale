# Skills

Bổ sung skills khi phát hiện pattern lặp lại trong quá trình code.

Thư mục này sẽ chứa các skill files (`.md`) mô tả pattern tái sử dụng sau khi code 2-3 features.

## Khi nào tạo skill?

- Khi thấy pattern code lặp lại 2-3 lần
- Khi có quy trình chuẩn cần document
- Khi cần template cho component/action mới

## Cấu trúc skill file

```markdown
# Skill Name
## Trigger: Khi nào activate skill này
## Steps: Các bước thực hiện
## Template: Code template nếu có
```
